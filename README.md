An NLP machine learning project to classify spam emails using TF-IDF, SVM, and Logistic Regression.

# Email Spam Classification using NLP

## 📌 Project Overview
This project is an intelligent classification system designed to detect and filter spam emails. It leverages Natural Language Processing (NLP) techniques and Machine Learning algorithms to accurately categorize text data as either "Spam" or "Ham" (Not Spam).

## 🛠️ Tech Stack & Technologies Used
* **Language:** Python
* **NLP Techniques:** NLTK, Tokenization, Stemming
* **Feature Extraction:** TF-IDF Vectorization
* **Machine Learning Models:** Support Vector Machine (SVM), Logistic Regression
* **Libraries:** Scikit-Learn, Pandas, NumPy

## ⚙️ Key Features & Workflow
1. **Data Preprocessing & Cleaning:** Handled raw dataset text, removed stop words, and applied stemming to normalize the text data.
2. **Feature Engineering:** Implemented Term Frequency-Inverse Document Frequency (TF-IDF) to convert textual data into numerical vectors suitable for machine learning models.
3. **Model Training & Tuning:** Trained and evaluated multiple classification models (SVM and Logistic Regression) to determine the most accurate predictor.
4. **Performance Evaluation:** Compared models based on accuracy and classification metrics to ensure high reliability in identifying spam.

## 🚀 Purpose
This project demonstrates practical skills in text processing, feature extraction, and deploying classical machine learning algorithms to solve real-world natural language processing problems.
